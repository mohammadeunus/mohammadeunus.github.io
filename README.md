# Getting started with Hugo??
1. install hugo windows. follow instruction link => https://gohugo.io/installation/windows/  
    - install chocochocolatey
    - install git
    - install Go
    - Download hugo prebuilt binary => copy the hugo.exe to program file/Hugo => Add this directory to the PATH environment variable.
    - install `hugo-extended` => choco install hugo-extended
2. go to theme directory, run hugo server
